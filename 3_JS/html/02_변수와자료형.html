변수 (variable) 

            Variable = Flexible object storage for memory
            변수에 이름을 붙이고 원하는 겂을 대입하면
            필요한 곳에서 변수명을 호출하여 저장된 값을 사용할 수 있음

            

                변수명 명명 규칙
            

            - 의미 있는 이름을 작성
            - 대,소문자 구분
            - 특수 문자 _ 만 사용 가능 (단, 첫글자 제외)
            - 숫자도 사용 가능 (단, 첫글자 제외)
            - 카멜 표기법으로 작성 "helloWorld"

            [Camel Case]
            문장, 여러 단어를 변수명으로 작성할 때 이어지는 첫 번째 단어를 대문자로 작성

Javascript 변수의 종류와 특징 

            var : variable, variable 중복 O (덥어쓰기), 함수 레벨 scope
            let : variable, variable 중복 X, 블록{} 레벨 scope
            const : constant, variable 중복 X, 블록{} 레벨 scope

            흔하게 쓰는 
            1순위 : const(JS는 특정 요소를 선택해서 사용하는 경유가 많기
                        때문에 const에 고정시켜줌)
            2순위 : let()

            3순위 : var()

Javascrip 자료형 

            JS 변수 선언시 별도의 자료형을 지정하지 않음
            -> 변수에 대입되는 값(literal)에 의해서 자료형이 결정됨

            - string (문자)
            - number (숫자)
            - boolean (논리) -> True/False
            - object (객체) -> Array, {k:v, k:v ..}
            - function (함수)
            - undefined (정의되지 않은 변수)

            -null (값 literal) : 참조하는게 없음
