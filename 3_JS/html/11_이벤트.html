이벤트(Even) 

            Even : 동작.행위
            -> 브라우저에서 할 수 있는 동작

            ex) click, keyup,keydown, keypress, hover, mouseover, mouseon, mouseout,
            mouseleave, drage, change, submit...

            * event listener
            -> 브라우저에서 이벤트가 감지 될 때까지 대기하다가 감지가 되었을때
            연결된 기능(함수)을 수행하는것 

            - event listener "on event name"
            ex) onclick, onchange, onsubmit


            * event handler
            -> a function of event listener (handler)
